## Úkoly

vytvořte funkce:
1. secti(a, b) - sečte a vrátí hodnoty.
2. odecte(a, b) - odečte hodnoty a vrátí.
3. nasob(a, b) - vynásobí hodnoty a vrátí.
4. del(a, b) - vydělí hodnoty a vrátí.

Vytvořte kalkulačku, základ programu máte v příloze index.js
